# TCA – Tricharmé Control Authority  
**Official DataHub Mirror – Version 2.0.0**

This space hosts the public DataHub mirror of the **Tricharmé Control Authority (TCA)** framework.  
It provides verified access to all semantic manifests, governance datasets, and controlled vocabularies used for:

- AI indexing  
- Knowledge graph population  
- Brand governance validation
